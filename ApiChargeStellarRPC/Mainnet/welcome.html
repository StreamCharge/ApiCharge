🐍 Python Example - Fee-Sponsored USDC Payment 

# Install: pip install stellar-sdk requests
import requests
from stellar_sdk import Keypair, Network, TransactionBuilder, Asset

SERVER = "https://mainnet.stellar.apicharge.com"
USDC = Asset("USDC", "GA5ZSEJYB37JRC5AVCIA5MOP4RHTM335X2KGX3IHOJAPP5RE34K4KZVN")

# 1. Get quote and purchase access token (see full example)
token = purchase_access_token(quote, keypair)

# 2. Build zero-fee transaction
tx = TransactionBuilder(account, network_passphrase=Network.PUBLIC_NETWORK_PASSPHRASE, base_fee=0)
    .append_payment_op(recipient, USDC, "1.00")
    .build()
tx.sign(keypair)

# 3. Submit - ApiCharge sponsors the fee!
requests.post(f"{SERVER}/apicharge/stablecoin/submit-transaction",
    headers={"apicharge": token}, json={"transactionXdr": tx.to_xdr()})

How it works:

  1. Purchase an access token by signing a USDC payment to the server
  2. Build your Stellar transaction with fee=0
  3. Submit via /apicharge/stablecoin/submit-transaction
  4. ApiCharge wraps it in a fee-bump and submits to the network

For standard RPC calls:

Purchase an RPC access token and use the /soroban/developer/ pass-through route with the apicharge header for all your Stellar JSON-RPC calls.
